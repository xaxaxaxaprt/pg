# xaxaxaxaprt Plugins

Revenge/Bunny için plugin koleksiyonu.

## Kurulum

Plugin URL'sini Revenge'in Plugins sayfasına yapıştır:
```
https://xaxaxaxaprt.github.io/pg/<PLUGIN_NAME>
```

## Pluginler

| Plugin | Açıklama |
|--------|----------|
| [MoreAlts](https://xaxaxaxaprt.github.io/pg/MoreAlts/) | 5'ten fazla hesap kullan, status picker'dan hesap değiştir |
| [ReadAll](https://xaxaxaxaprt.github.io/pg/ReadAll/) | Tüm bildirimleri tek komutla oku - /readall |
| [MessageScheduler](https://xaxaxaxaprt.github.io/pg/messageScheduler/) | Mesajları zamanla |
| [NSFWBypass](https://xaxaxaxaprt.github.io/pg/nsfwbypass/) | NSFW kanallarına yaş ayarlamadan eriş |

## MoreAlts Özellikleri

- Sınırsız hesap kaydetme
- Status picker'dan hızlı hesap değiştirme (You butonuna basılı tut)
- Settings sidebar'da Account Switcher
- Hesap export/import (şifreli)
- CLI komutları: `/accswitcher add`, `/accswitcher login`, `/accswitcher list`
